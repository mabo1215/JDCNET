\section*{Overall Evaluation}

This paper is most credible as a reproducible pilot study rather than as a new-method submission. The current draft already has many of the right structural elements: patient-level paired evidence, a fixed-split feasibility screen, repeated same-case resampling, and explicit evidence-role tables. The remaining work is therefore final positioning, not new science.

\section*{Major Strengths}

\begin{enumerate}
    \item Clear problem focus on training-only CT supervision for X-ray deployment under tiny patient-level paired cohorts.
    \item Strong evaluation hierarchy with fixed-split feasibility screening and repeated patient-level resampling as the primary evidence layer.
    \item Good use of evidence-role captions and support counts that make the limitations of the paired cohort visible.
    \item A reproducibility-minded appendix that documents split audits, per-seed instability, imbalance checks, and ablation diagnostics.
\end{enumerate}

\section*{Priority 1: Reinforce Pilot Framing in High-Visibility Locations}

\textbf{Current issue:}

The paper already reads much more like a pilot study than a method paper, but a few top-line locations still leave room for a reader to interpret it as an architecture validation submission.

\textbf{Revision action:}
\begin{enumerate}
    \item Emphasize the evidence protocol and deployment-oriented task in the title and abstract, and avoid phrasing that elevates JDCNet into the main claimed contribution.
    \item In the first method-facing sentence of the abstract and the opening of Section~3, present JDCNet as an executable scaffold for testing hypotheses rather than as a validated design.
    \item Move lower-priority mechanism details (DPE/MHRA/DFPN) further toward the appendix if needed, so the main text stays centered on the evaluation design.
\end{enumerate}

\section*{Priority 2: Keep the Main Headline Table Focused on Same-Case Paired Evidence}

\textbf{Current issue:}

A CT-only reference row in the main paper risks competing visually with the paired-cohort evidence, even when it is properly labeled as a non-comparable baseline.

\textbf{Revision action:}
\begin{enumerate}
    \item Remove the CT-only reference row from the main headline fixed-split table.
    \item If needed, report the CT-only result only in the appendix or in a short note immediately below the table, explicitly labeling it as a data-availability reference.
    \item Keep the main text discussion focused on rows that materially support the paired-cohort argument.
\end{enumerate}

\section*{Priority 3: Anchor the Opening Related Work in Broader Thoracic Imaging}

\textbf{Current issue:}

The first related-work subsection is still slightly narrow, with too much emphasis on COVID-specific model papers and not enough on canonical chest X-ray benchmarks or general thoracic imaging evaluation standards.

\textbf{Revision action:}
\begin{enumerate}
    \item Add one broader chest X-ray benchmark/dataset anchor such as NIH ChestX-ray14, MIMIC-CXR, or another well-cited thoracic imaging benchmark.
    \item Optionally add one more high-level medical imaging survey or evaluation-standard citation.
    \item Use these references to frame the opening subsection as general chest imaging context rather than COVID-only prior work.
\end{enumerate}

\section*{Priority 4: Trim Redundant Fixed-Split Text}

\textbf{Current issue:}

The fixed-split narrative now contains the right caution, but it still repeats points that are already explicit in the revised table caption and figure caption.

\textbf{Revision action:}
\begin{enumerate}
    \item Remove one or two sentences that restate the same caution about the four-image paired split, the all-positive tendency, and one-negative validation case.
    \item Preserve the essential sentence that explains why the fixed split is a feasibility screen and not a definitive ranking.
    \item Keep the reader focused on the repeated-resampling table as the primary evidence layer.
\end{enumerate}

\section*{Non-Priority Guidance}

\begin{enumerate}
    \item Do not add another experiment unless it materially increases the same-case evidence scale (more paired patients or an independent paired cohort).
    \item Do not broaden the claim beyond ``training-only CT supervision for X-ray deployment under tiny patient-level paired cohorts.''
    \item Do not re-expand the method description in Section~3; keep the main text lean and shift lower-confidence details to the appendix.
\end{enumerate}
