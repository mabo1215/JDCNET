\section*{Overall Evaluation}

当前稿件相较于上一轮已经完成了最关键的方向修正：problem formulation 已统一，claim boundary 已明显收紧，fixed split 与 repeated same-case resampling 的证据层级已经区分清楚，negative/fragile finding 也被诚实写出。主文和附录现在基本形成了一个较为一致的叙事：这不是一篇已经验证了新架构优越性的论文，而是一篇 \textbf{evidence-bounded reproducible pilot study}。

从当前 `paper/main.tex`、`paper/appendix.tex` 与已编译 PDF 来看，稿件最重大的结构性问题已经不再是“定义不清”或“过度宣称”，而是剩下三类更细但依然重要的收口工作：

\begin{enumerate}
    \item \textbf{方法篇幅仍略重，和当前 pilot-paper 定位相比还有一点 architecture-heavy。}
    \item \textbf{主文对 evaluation support counts 的可见性还可以继续增强，让 reviewer 扫图表时也不容易误判证据规模。}
    \item \textbf{Related Work 和 scholarly positioning 还有继续补强空间，尤其是 privileged modality / cross-modal supervision transfer / small-cohort evidence robustness 三条线。}
\end{enumerate}

因此，当前稿件的判断更适合写成：

\[
\boxed{\text{Current recommendation: still major revision for a method-style submission,}}
\]
\[
\boxed{\text{but already much closer to a credible pilot-study / reproducibility paper.}}
\]

\section*{Major Strengths of the Current Draft}

\begin{enumerate}
    \item \textbf{问题定义终于稳定。} 当前主文已经基本杜绝了把本文误读成“test-time multimodal fusion”或“validated new architecture paper”的风险。
    \item \textbf{证据层级明显改善。} fixed split 现在只是 feasibility screen，而 repeated same-case resampling 才是 primary evidence，这一点是正确方向。
    \item \textbf{负结果写得足够诚实。} 稿件现在愿意明确承认 plain control、same-modality KD、late fusion 在某些指标上比 full JDCNet 更强，这比硬包成正结果更有学术可信度。
    \item \textbf{附录已经有实质价值。} split audit、per-seed instability、threshold behavior、progressive complexity、error analysis 等补充材料，确实在帮助界定“哪些话能写、哪些话不能写”。
\end{enumerate}

\section*{Priority 1: Rebalance Method Emphasis Against Evidence Emphasis}

\textbf{当前问题：}
Section 3 虽然已经加入了 hypothesis-driven framing，但从篇幅体感上看，DPE/MHRA/DFPN 的方法展开仍略重。对于一篇已经明确转向 pilot study 的稿件，过多的模块级公式和局部设计动机，仍会让 reviewer 下意识地用“新方法论文”的标准来读这篇文章。

\textbf{建议：}
\begin{enumerate}
    \item 继续压缩主文中 3.2--3.9 的 architecture-forward 叙事，尤其是那些对当前 claim boundary 没有新增信息量的重复性表述。
    \item 若版面吃紧，优先把较低信号的模块细节或公式说明移到 appendix，而不是压缩 evidence-boundary discussion。
    \item 主文方法部分应保留“可复现所必需的定义”，但不要再给人一种“本文主要贡献是提出并验证了一整套新模块”的视觉印象。
\end{enumerate}

\section*{Priority 2: Make Evaluation Support Counts Impossible to Miss}

\textbf{当前问题：}
正文已经写出 fixed split 只有 4 张验证图像、resampling 每次只有 1 个 negative validation patient，但这些关键信息仍主要散落在正文段落里。对只扫图表和 caption 的 reviewer 来说，证据规模仍有被低估或误读的风险。

\textbf{建议：}
\begin{enumerate}
    \item 在主文中增加一个更紧凑的 evaluation-regime summary，可用小表、注释框或更强的 caption 文字来完成。
    \item 明确把以下信息并排可见地写出来：evaluation regime、patient counts、image counts、positive/negative support、以及该 regime 在论文中的角色（reference / feasibility / primary evidence）。
    \item 若不想再加新表，至少应继续强化固定 split 主表和 resampling 主表 caption 中的 support-count 提示。
\end{enumerate}

\section*{Priority 3: Strengthen Scholarly Positioning and References}

\textbf{当前问题：}
Related Work 已经比之前清楚得多，但若按目标期刊标准，仍可以再补 2--4 篇更强、更“定锚”的文献，使本文的学术位置更稳，而不只是“现在比上一轮更合理”。

\textbf{建议重点：}
\begin{enumerate}
    \item privileged modality / learning using privileged information 在医学影像或相邻领域中的后续发展；
    \item cross-modal supervision transfer / representation transfer 的更强 canonical work；
    \item small-cohort medical imaging evaluation、reproducibility、evidence inflation 相关文献。
\end{enumerate}

\textbf{写法上还应继续强调：}
\begin{itemize}
    \item 本文\textbf{没有}做全面的 feature-alignment design-space search；
    \item 本文\textbf{没有}给出 clinical-grade external validation；
    \item 本文\textbf{没有}验证“test-time multimodal system”，而是在研究 training-only auxiliary modality。
\end{itemize}

\section*{Priority 4: Keep Headline Tables Maximally Interpretable}

\textbf{当前问题：}
主文 headline table 已经比之前更稳，但未来再收口时仍要防止“不可同案比较的参考行”和“paired same-case evidence”在视觉上过于接近。

\textbf{建议：}
\begin{enumerate}
    \item 保持所有 non-same-case comparable rows 都明确标记为 \texttt{reference}。
    \item 如果版面需要继续瘦身，可优先考虑把最容易引起误读的 CT-only reference row 移到 appendix，仅在主文保留 paired-cohort 的核心比较。
    \item 所有 headline figures/tables 都应优先服务于“证据边界清晰”，而不是“结果看起来更满”。
\end{enumerate}

\section*{Priority 5: If Another Experiment Is Run, Only Run One That Changes the Evidence Scale}

\textbf{当前问题：}
在现有逻辑下，再做 fixed-split micro-ablation、更多 threshold slicing、或更多同一小 cohort 上的局部小改动，已经很难继续增加真正的 evidential strength。

\textbf{下一步真正值得做的实验只剩两类：}
\begin{enumerate}
    \item 更大的 patient-level paired cohort；
    \item 在\textbf{同样的 repeated same-case resampling protocol} 下补一个更强的 generic feature-alignment baseline。
\end{enumerate}

\textbf{不建议：}
\begin{enumerate}
    \item 再用当前 4-image fixed split 做更多主结论；
    \item 把更多微型调参包装成“实验补强”；
    \item 在没有真正任务匹配和独立性的情况下，硬写一个伪 external validation。
\end{enumerate}
